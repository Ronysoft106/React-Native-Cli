NextJobzApp is a modern job search mobile application designed to help users find jobs near them and explore top companies. The app features:

A clean and intuitive UI inspired by popular job platforms.

Search functionality for jobs and companies.

A dynamic home screen with job listings and company highlights.

A sleek bottom tab bar for easy navigation between Home, Profile, Jobs, Dashboard, and More sections.

Attractive job cards with company logos, salary info, job tags, and an easy "Apply now" button.

Responsive design with smooth horizontal scrolling for jobs and companies.

Built using React Native CLI for a performant native experience.
